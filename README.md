# 🏕️ Força Acamp 2026 — Sistema de Inscrição

> Aplicação web desenvolvida para gerenciamento de inscrições do evento **Força Acamp 2026**, com foco em organização, praticidade e experiência do usuário.

---

## 📌 Sobre o projeto

Este projeto tem como objetivo facilitar o processo de inscrição de participantes em eventos, centralizando dados e tornando o fluxo mais simples e eficiente.

A aplicação foi construída com foco em performance, organização e escalabilidade, podendo ser adaptada para diferentes tipos de eventos.

---

## 🚀 Funcionalidades

- 📝 Cadastro de participantes
- 📋 Formulário de inscrição online
- 🔎 Organização e estrutura de dados
- ⚡ Interface rápida e responsiva
- 🔄 Estrutura pronta para integração com backend
